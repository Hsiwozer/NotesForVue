# vite 和 vue-cli 的比较

|                      |     vite      |     vue-cli     |
| :------------------: | :-----------: | :-------------: |
| **支持的 vue 版本**  | 仅支持 vue3.x | 支持 3.x 和 2.x |
| **是否基于 webpack** |      否       |       是        |
|     **运行速度**     |      快       |      较慢       |
|    **功能完整度**    |    小而巧     |     大而全      |
